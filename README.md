# ⚡ Smart Cache Cleaner (Simulation Mode)

### ⚠️ IMPORTANT NOTICE
Please note that this software is currently in an **EXPERIMENTAL SIMULATION (DEMO) PHASE**. 
It calculates the exact junk files on your PC and shows the graphical report, but **it does NOT modify or delete any files** from your system. It is 100% safe to test.

---

A lightweight, high-speed, and open-source system analyzer built with Python. It scans temporary files and estimates system cache recovery without touching your personal data or browser passwords.

## ✨ Features
* **High-Speed Scan:** Analyzes thousands of temporary files in milliseconds.
* **Graphical Progress Bar:** Displays a live, real-time loading bar showing the scanning progress.
* **100% Privacy & Safety:** Runs entirely offline on your local computer. No data tracking.
* **Zero Antivirus Blocks:** By running the raw source code directly, Windows Smart App Control and antiviruses will not block the software.

## 🚀 How to Run

To ensure maximum safety and transparency, this software runs directly from its Python source code. This bypasses rigid Windows restritions (like Smart App Control) since you can see every single line of code.

### Step 1: Install Python
1. Open the **Microsoft Store** on your Windows PC.
2. Search for **Python** (choose version 3.11, 3.12, or the latest available).
3. Click **Get** / **Install**. It is official, free, and sets up in seconds.

### Step 2: Download and Run the Script
1. Download the `cleaner.py` file from this repository to your computer.
2. Go to the folder where you saved the file.
3. **Double-click** `cleaner.py` to run it instantly!

*Alternative for developers (from terminal):*
Open your Command Prompt (`cmd`), navigate to the file directory, and type:
```bash
python cleaner.py
```
