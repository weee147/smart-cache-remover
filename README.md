# ⚡ Smart Cache Cleaner

### ⚠️ IMPORTANT NOTICE
Please note that this software is still in an **EXPERIMENTAL (BETA) PHASE**. 
While it is designed with safety features like the Desktop Quarantine folder, **100% functionality and stability are NOT guaranteed**. Use it at your own risk.

---

A lightweight, high-speed, and open-source junk file cleaner built with Python. It safely removes temporary files and clears system cache without touching your personal data or browser passwords.

## ✨ Features
* **High-Speed Scan:** Scans and analyzes thousands of files in milliseconds.
* **100% Privacy:** Runs entirely offline on your local computer. No cloud servers, no data tracking.
* **Safety Quarantine:** Moves uncertain or older files to a temporary desktop folder first, so you can double-check them before permanent deletion.
* **Optional Recycle Bin Emptying:** Prompts for confirmation to empty the system trash at the very end.

## 🚀 How to Run
1. Download `cleaner.py` to your computer.
2. Make sure you have Python installed on your system.
3. Run the script by double-clicking it or using your terminal:
   ```bash
   python cleaner.py
   ```
