# Contributing to Smart Cache Cleaner 🤖

Thank you for your interest in improving this project! We love community contributions and want to make the development process as smooth as possible.

## 🐛 How to Report Bugs & Issues
If you find a bug (e.g., a crash during cleanup or an unlocked file error):
1. Go to the **Issues** tab on GitHub.
2. Click **New Issue**.
3. Describe the problem clearly: what happened, what game/cache folder caused it, and your Windows version.

## 💡 Suggesting New Features
Want to add support for a new game cache (like *Valorant*, *League of Legends*, or *Steam*)? 
* Open an issue labeled **Enhancement** or **Feature Request** to discuss your idea before writing code.

## 🛠️ Development Guidelines
If you want to modify the code directly or help build the **v2.0 Ultimate Edition**:
1. **Fork** this repository.
2. Create a new branch for your feature (`git checkout -b feature/AmazingFeature`).
3. Keep the user interface uniform by using the `customtkinter` library.
4. Ensure all code is properly commented and clean.
5. Open a **Pull Request (PR)** explaining your changes clearly.

## 📜 Code of Conduct
Please be respectful and kind to other contributors in all discussions, issues, and pull requests. Let's build a helpful tool together!
